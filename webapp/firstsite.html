Control Application for LEDs switching
For the purpose of ELEC3215 Coursework

Instructions

There are three slave boards in the network.
You can control the LED on each board.
LED can illuminate in three different colours: red, green and blue.
You can control each slave independently.
Only one LED colour can be set for the board at the time.
